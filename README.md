📊 Data Analyzer

Data Analyzer is a simple full-stack web application that allows users to upload CSV files and automatically generate insights, summaries, and visualizations—without writing any code.

🚀 Features
      📂 Upload CSV datasets
      📊 Automatic data summary
      Number of rows & columns
      Column names
      Missing values
      Mean of numeric columns
      📈 Data visualization using charts
      🔗 Frontend–Backend integration
      ⚡ Fast and beginner-friendly
  🛠️ Tech Stack
          Backend:
          Python
          Flask
          Pandas
          Frontend:
          HTML
          JavaScript
          Chart.js
